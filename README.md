# Telegram Assistant Bot 🤖

Telegram-бот с пользовательским интерфейсом, обработкой данных и админ-панелью.
Бот: @xvoltar_bot

## 🚀 Функционал

- Inline-кнопки (интерактивное меню)
- Сбор данных пользователей (ID, username, телефон)
- Хранение данных в SQLite
- Проверка на дубли
- Админ-панель:
  - просмотр пользователей (/users)
  - рассылка сообщений (/broadcast)
- Калькулятор

## 🛠 Стек

- Python
- pyTelegramBotAPI
- SQLite
