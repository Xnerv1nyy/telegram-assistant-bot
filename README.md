# Telegram Assistant Bot 🤖

Полноценный Telegram-бот с интерфейсом, базой данных и админ-панелью.

## 🚀 Возможности

* 📱 Сбор данных пользователей (ID, username, телефон)
* 💾 Хранение в SQLite
* ❌ Защита от дублей
* 🧮 Калькулятор
* ⚡ Inline-кнопки (UI как у сервисов)
* 👑 Админ-панель:

  * /users — список пользователей
  * /broadcast — рассылка

## 🛠 Стек

* Python
* pyTelegramBotAPI
* SQLite

## 📸 Демонстрация

Бот поддерживает:

* интерактивное меню
* обработку данных
* администрирование пользователей

## 📦 Установка

```bash
pip install -r requirements.txt
python bottg.py
```

## ⚙️ Конфигурация

Создай `.env` файл:

```
TOKEN=your_token
ADMIN_ID=your_id
```
